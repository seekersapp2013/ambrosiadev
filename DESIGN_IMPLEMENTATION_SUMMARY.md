```
Deep Rose (#BF1A65)    - Brand identity, primary actions
Pink (#ec4899)         - Hover states, highlights
Purple (#8B5CF6)       - Secondary actions, mindfulness

```
Success: Green (#10B981)    - Health, growth, confirmation
Warning: Amber (#F59E0B)    - Alerts, premium content
Error: Red (#EF4444)        - Errors, critical actions
Info: Blue (#3B82F6)        - Information, trust

Border: #E5E7EB (Gray 200)

## 🎯 Design Principles

### 1. Mobile-First Approach
- Design for 375px width minimum
- Touch targets minimum 44x44px
- Thumb-friendly bottom navigation
- Swipe gestures for natural interactions

### 2. Health & Wellness Aesthetic
- Clean and minimal interface
- Soft, approachable design language
- Trustworthy and professional
- Energetic and motivating
- Accessible to all users

### 3. Content-Focused
- Clear visual hierarchy
- Generous white space
- Readable typography
- High-quality imagery
- Distraction-free reading experience

### 4. Accessibility Standards
- WCAG 2.1 Level AA compliance
- 4.5:1 contrast ratio for text
- Keyboard navigation support
- Screen reader optimization
- Reduced motion support

## 📱 Key Screen Highlights

### Authentication Flow
1. **Welcome Screen**: Gradient background, compelling value proposition
2. **Sign Up**: Clean form with email authentication
3. **Interest Selection**: Visual grid of health topics, unlimited selection

### Home Experience
1. **Article Feed**: Instagram-style cards with engagement metrics
2. **Story Tags**: Horizontal scroll for quick topic navigation
3. **Premium Indicators**: Clear lock icons and pricing for gated content

### Booking System
1. **Provider Browser**: Grid layout with dual pricing (1-on-1 and group)
2. **Calendar Interface**: Interactive date/time selection
3. **Event Cards**: Group sessions with participant counts and availability
4. **Provider Dashboard**: Comprehensive booking management

### Wallet & Payments
1. **Balance Display**: Gradient card showing USD and CELO amounts
2. **Transaction History**: Clear list with icons and timestamps
3. **Paywall**: Compelling unlock screen with benefits list
4. **Fund Wallet**: QR code and address for easy deposits

### Profile & Social
1. **Profile Header**: Stats, bio, interests preview
2. **Comprehensive Interests**: Collapsible analytics sections
3. **Content Tabs**: Posts, saved, followers, following, interests
4. **Edit Profile**: Simple form with photo upload

### Messaging
1. **Chat List**: Unread indicators, timestamps, avatars
2. **Chat Window**: Bubble design with typing indicators
3. **Privacy Settings**: Granular control over who can message

### UI Patterns
- **Bottom Sheets**: Mobile-friendly modals
- **FAB**: Floating action button for primary actions

## 🎭 User Journey Integration

### 1. New User Onboarding
```
Welcome → Sign Up → Interest Selection → Profile Setup → Home Feed
```
**Design Focus**: Welcoming, educational, quick setup

### 2. Content Discovery
```
Home Feed → Article Card → [Free/Premium] → Reader/Paywall
```
**Design Focus**: Easy browsing, clear pricing, smooth unlocking

### 3. Expert Booking
```
Booking Browser → Provider Card → Calendar → Payment → Confirmation
```
**Design Focus**: Trust-building, clear availability, secure payment

### 4. Content Creation
```
Home → Create Button → [Article/Reel] → Editor → Publish
```
**Design Focus**: Simple tools, preview options, monetization settings

### 5. Wallet Management
```
Wallet → Balance View → [Fund/Transfer] → Transaction Complete
```
**Design Focus**: Clear balance, easy funding, transaction history
